# kakarot-repurposer

把一篇写好的公众号长文，一稿改写成**小红书笔记** + **抖音脚本**。

保留「卡卡罗特学AI」的活人感，但按各平台调性重新表达——不是复制粘贴，也不是机械缩写。

**触发：** 对 Claude 说「把这篇转成小红书」「改成抖音脚本」「一稿多平台」。

**输出：** 小红书笔记版（标题+正文+话题标签+配图建议）、抖音脚本版（一句一行的分句脚本，纯文字/AI配音/真人念三种用法通用）。

**配合：** kakarot-writer 写长文 → kakarot-repurposer 改多平台 → wechat-publisher 发公众号草稿箱（小红书/抖音手动或第三方工具发）。

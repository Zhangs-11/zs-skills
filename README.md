# zs-skills

Kakarot 的个人 Claude Code Skill 合集。

## 目录

- [kakarot-writer](./kakarot-writer/) — Kakarot说AI 公众号长文写作 skill
- [aihot](./aihot/) — AI HOT 每日资讯查询 skill（源自 [khazix-skills](https://github.com/KKKKhazix/khazix-skills)）
- [system-structure-diagram-skill](./system-structure-diagram-skill/) — 系统架构图生成 skill
- [wechat-publisher](./wechat-publisher/) — 公众号文章发布工具（配合 kakarot-writer 使用）

## 安装

在 Claude Code 中直接输入：

```
帮我安装这个 skill：https://github.com/Zhangs-11/zs-skills/tree/main/<skill-name>
```

将 `<skill-name>` 替换为 `kakarot-writer`、`aihot`、`system-structure-diagram-skill` 或 `wechat-publisher`。

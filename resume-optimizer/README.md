# resume-optimizer — 程序员简历优化 Skill

以资深技术面试官的视角，帮开发者写出能拿到面试邀约的简历。

## 能做什么

- **从零写简历**：给出经历素材，按标准结构产出完整简历
- **整体优化**：现有简历全文重组、逐模块打磨
- **单模块打磨**：只改项目经验、专业技能等某一块
- **简历评审**：指出亮点、问题和逐条修改建议

## 核心特点

- **绝不编造**：只重组和提炼你给的真实信息，缺什么会直接问你，不脑补经历、项目和数据
- **面向筛选场景**：按「简历是广告不是档案」的思路写，每一行都为拿到面试服务
- **按年限分层**：3 年 / 5 年 / 8 年经验的简历侧重点完全不同，规范里分别定义
- **成果导向**：动作描述一律改写为可量化的成果描述

## 安装

复制本目录到 Claude Code 的 skills 路径：

```bash
cp -r resume-optimizer ~/.claude/skills/resume-optimizer
```

无额外依赖，不需要 API Key。

## 怎么用

安装后直接对 Claude 说：

- "帮我写一份简历，我的经历是……"
- "优化一下我的简历"（附上简历内容或文件）
- "帮我把项目经验这块改好一点"
- "以面试官视角评审下这份简历"

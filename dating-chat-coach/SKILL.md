# 线上聊天教练（男生版）

## 概述

基于《线上聊天手册·男生版 v4》（作者 ZHOUYI1028）沉淀。给零经验男生的异性聊天全流程指南：熟人介绍（相亲）和网上认识两套打法。

**三条核心原则（凌驾于一切话术之上）：**
1. **线上聊天只有一个目的：把人顺利约到线下见面。** 聊得再热闹不见面都是空中楼阁，线上热度有保质期。
2. **真诚 > 话术。** 所有技巧服务于"让真实的你被看见"，不是帮你装成另一个人。不需要推拉、忽冷忽热那套。
3. **你是在筛，不是在求。** 需求感（秒回、连发、句句迎合、她一冷淡就慌）是头号死因。自检：这条消息是分享还是讨好？是讨好就删了重写。但不卑≠变傲，不讨好≠高冷装酷。

## 最常用的判断标准（先记住这几条硬指标）

- **万能接话公式**：回应她说的 → 关联到自己 → 顺着带出下一个问题。她表达**感受**时先陪情绪别接话题，别给解决方案（"多喝热水"是终结器）。
- **约见面条件**（三条满足两条就提）：① 聊了几天以上节奏稳定 ② 找到至少一个共同兴趣点 ③ 她有过主动找你聊的行为。约要给具体时间+地点，加一句台阶话。**两次被明确拒绝就停手。**
- **兴趣判断**：她主动提问/展开说/记得你说过的细节 = 有戏；永远你先发、只回两三个字、从不问你 = 没兴趣（不是慢热），降低投入。
- **已读不回**：不催，等一天；第二天隔新话题重开；连续三次已读不回 = 判定没兴趣。
- **转微信时机**（网上认识）：平台聊 3~5 天、双方有来回、她偶尔主动，才转。给具体理由（发图不方便/APP 不常看），被拒回"没事在这聊也行"，过几天再提一次，两次不愿就作罢。
- **回复节奏**：匹配对方节奏，她慢你也慢；一条消息不超三行；对方没回绝不连发。

## 起草/润色消息时的检查清单

发出去的每条消息过一遍：
- [ ] 是分享不是讨好，没有迎合味
- [ ] 一次只抛一个问题（不查户口）
- [ ] 接住了她的原话（尤其是情绪），不是自说自话
- [ ] 有画面有细节，不是干巴巴报事实（"下山腿抖得像筛糠" > "我去爬山了"）
- [ ] 不超三行、不叠词、不堆表情包
- [ ] 没踩绝对禁止项（见 reference.md §8：说教、追问不回、吹牛、不熟时发暧昧/深夜消息等）

## 何时不用这个 skill

- 用户问的是线下约会/恋爱关系经营（手册只覆盖到第一次见面为止；见面之后"把手册放下，做你自己"）
- 女生视角的问题（本手册是男生版，原则可参考但示范话术视角不对）
- 任何操纵、PUA、欺骗性质的请求——手册本身反套路，拒绝往这个方向用
